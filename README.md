## Academic Coursework & Featured Projects

### Course: Deep Learning (ITAI 2376)
**Overview:** An intensive, project-driven course covering the end-to-end lifecycle of deep learning model development, from foundational neural networks to advanced generative AI and reinforcement learning agents.

#### 1. [Midterm Project: Diffusion Model from Scratch](./DeepLearning-ITAI2376/Midterm_Diffusion_Model/)
Designed and trained a probabilistic diffusion model capable of generating high-fidelity images from pure noise. The project involved implementing a U-Net architecture with time-step embeddings, highlighting a deep understanding of the mathematical foundations behind state-of-the-art image generation tools.
- **Key Technologies:** PyTorch, Torchvision, Einops, Matplotlib
- **Highlights:** Implemented forward diffusion (noise addition) and reverse diffusion (denoising) processes.

#### 2. [Reinforcement Learning: Autonomous CartPole Agent](./DeepLearning-ITAI2376/Lab09_RL_CartPole/)
Developed an autonomous Reinforcement Learning agent capable of balancing a pole on a moving cart within a physics simulation. Transitioned the agent from a randomized baseline to a highly performant Q-Learning policy that dynamically balances exploration and exploitation.
- **Key Technologies:** Gymnasium, Q-Learning, Epsilon-Greedy Policies, NumPy
- **Highlights:** Successfully trained the agent over 500 episodes to consistently achieve the maximum possible survival score, demonstrating the core principles behind RLHF.

#### 3. [Sequence Modeling: RNNs & Transformers](./DeepLearning-ITAI2376/Lab05_RNN_Transformers/)
Explored sequential data modeling by implementing Recurrent Neural Networks (RNNs) and modern Transformer architectures, focusing on natural language processing tasks and attention mechanisms.
- **Key Technologies:** TensorFlow, Keras, Attention Mechanisms

#### 4. [Computer Vision: Convolutional Neural Networks](./DeepLearning-ITAI2376/Lab03_CNNs/)
Built, tuned, and evaluated CNN architectures for image classification tasks, applying techniques like pooling, dropout, and data augmentation to improve model generalization.
- **Key Technologies:** Convolutional Networks, Image Preprocessing, Feature Extraction

#### 5. [Neural Network Foundations](./DeepLearning-ITAI2376/Lab02_Neural_Networks/)
Constructed fundamental neural network architectures to understand backpropagation, activation functions, and gradient descent optimization from the ground up.
- **Key Technologies:** Deep Learning Foundations, Optimization Algorithms

#### 6. [NLP & Sequence Modeling Analysis](./DeepLearning-ITAI2376/Assignment04_NLP/)
Comprehensive analysis of sequence modeling approaches and the evolution of Natural Language Processing methodologies prior to the Transformer era.
